# Pacman Game (C++)

A console-based Pacman game developed in C++ using Windows Console APIs.

## Features

- Classic Pacman gameplay
- Five moving ghosts
- Score system
- High score saving
- Extra lives
- Collision detection
- Game Over screen

## Technologies Used

- C++
- Windows.h
- Console API
- File Handling

## Controls

| Key | Action |
|-----|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

## How to Run

1. Open the project in Visual Studio Code or any C++ IDE.
2. Compile the `pacman.cpp` file.
3. Run the generated executable.
